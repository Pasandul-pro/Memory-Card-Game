# 🧠 Memory Card Game

A sleek, modern, and interactive Memory Card Game built with **React 19** and **Vite**. Challenge your memory by matching pairs of emojis in this beautifully designed web application.

![Memory Card Game Preview](https://via.placeholder.com/800x450.png?text=Memory+Card+Game+Preview) <!-- Replace with actual screenshot if available -->

## ✨ Features

- **Responsive Grid**: A dynamic 4x4 grid of cards that adapts to any screen size.
- **Interactive Animations**: Smooth card flipping and matching animations powered by CSS 3D transforms.
- **Score Tracking**: Keep track of your matches and total moves to challenge yourself.
- **Win Detection**: A celebratory message appears when you successfully match all pairs.
- **Sleek UI**: Modern dark-themed design with glassmorphism effects and vibrant gradients.
- **One-Click Reset**: Easily restart the game at any time with the reset button.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Modern CSS features like Flexbox, Grid, and Animations)
- **Icons/Content**: Emojis 🍎🍌🍇

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pasandul-pro/Memory-Card-Game.git
   cd Memory-Card-Game
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173` to see the game in action!

## 🎮 How to Play

1. Click on any card to reveal the emoji behind it.
2. Click on a second card to find a match.
3. If the emojis match, the cards stay face up and turn green!
4. If they don't match, they flip back after a short delay.
5. Match all pairs in as few moves as possible to win!
